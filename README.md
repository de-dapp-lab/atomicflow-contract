# Atomicflow Contract

## Requirements

- Node: +18.12
  - Recommend using `nodenv` to automatically load the version

## Command

```shell
# Basics

yarn compile
yarn test
yarn format


# Analytics

# Get gas estimation
yarn test:gas
# Get contract size
yarn csize
```
